LANCER LE PROJET :
******************

Côté Back =
___________

-> Lancer le serveur Symfony :
        "symfony server:start --port 3001"

-> Ouvrir mariadb :
        "sudo service mariadb start"
        " mysql -u root -proot"
    Nom de la DB : "ecommerce"

Côté Front =
____________

-> Lancer le serveur React :
        "npm run dev"

-> Ouvrir le localhost:3000 sur le navigateur
npm

Côté API =
___________

-> Lancer l'API :
        E-ecommerce/api >
        "npm run start"
